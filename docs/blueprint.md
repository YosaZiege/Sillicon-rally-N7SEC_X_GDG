# **App Name**: Security Challenge Arena

## Core Features:

- Password Master Challenge: Generates passwords and provides real-time strength scoring with visual feedback and tips, reinforcing best practices.
- Phishing Email Detective: Presents users with simulated emails to identify phishing attempts, providing immediate feedback and highlighting red flags.
- Secret Message Decoder: Challenges users to decode cipher puzzles with hints and speed bonuses, enhancing their understanding of basic cryptography.
- Spot the Scam Quiz: Presents scenario-based multiple-choice questions to test users' ability to recognize scams, with personalized results and explanations.
- Digital Footprint Simulator: An interactive story that demonstrates how online choices affect privacy and security, providing a digital footprint report and personalized tips.
- Two-Factor Authentication Demo: Interactive simulation that illustrates the effectiveness of 2FA methods, followed by a quiz to reinforce understanding.
- Privacy Settings Puzzle: Simulates social media profile settings, challenging users to identify and fix hidden privacy issues within a set time limit.
- Social Engineering Challenge: Presents real-world social engineering scenarios in a role-play format, showing the consequences of each choice on a trust and security meter.
- Certificate Generation: Generates a downloadable certificate with the user's name, total score, and a 'Security Champion' title if the score is high enough.
- Leaderboard Management: Tracks and displays top 10 scores with names using browser localStorage, allowing users to add their score to the leaderboard after completion.

## Style Guidelines:

- Primary color: Blue (#2563eb) to convey trust and security.
- Accent color: Purple (#7c3aed) for highlights and calls to action.
- Success color: Green (#10b981) to indicate correct answers and challenge completion.
- Danger color: Red (#ef4444) to highlight incorrect answers and security risks.
- Body and headline font: 'Inter' (sans-serif) for a clean and accessible reading experience.
- Use a mix of security-themed emoji and simple SVGs to visually represent each challenge, instructions and feedback.
- Implement a card-based layout for displaying challenges on the homepage.
- Incorporate smooth hover effects, loading animations between challenges, and a confetti animation upon challenge completion.
